## Run Locally

Clone the Project repo

```bash
  git clone https://github.com/janvi01/MaidAid.git
```

Open the project on your IDE

```bash
  cd MaidAid
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
Head over to the http://localhost:3000/ to view it live.
