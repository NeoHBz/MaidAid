## Run Locally

Clone the project

```bash
  git clone https://github.com/janvi01/MaidAid.git
```

Go to the project directory

```bash
  cd MaidAid
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
